# 步骤 17 — URL 抓取功能

> **新增功能:** URL 抓取 `/fetch-url`

---

## 新增路由

| 路由 | 方法 | 说明 |
|------|:----:|------|
| `/fetch-url` | POST | 抓取用户提交的 URL 并显示响应内容 |

### 功能说明

- 需登录才能访问
- 从表单接收 `url` 参数
- 使用 `urllib.request.urlopen()` 访问目标 URL
- 返回状态码和前 5000 字符内容
- 超时时间 10 秒

### 新增/修改文件

| 文件 | 说明 |
|------|------|
| `app.py` | 新增 `/fetch-url` 路由 + `urllib` 导入 |
| `templates/index.html` | 新增 URL 输入框、抓取按钮、结果展示区 |
